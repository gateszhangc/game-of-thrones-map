## 项目开发规则

### 禁止的操作
- **不要自动编译项目** - 除非明确要求，否则不要运行 `npm run build`
- **不要重启服务端口** - 避免重新启动开发服务器 (npm run dev, npm start 等)
- **不要运行长时间的任务** - 避免运行可能阻塞开发流程的命令
- **不要自动启动任何服务** - 即使是为了测试，也要先询问用户
- 🚫**不要自动提交代码** - 仅在用户明确要求时才执行 git 提交
